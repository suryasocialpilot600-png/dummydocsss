The SocialPilot API allows you to manage team members, roles, and permissions within your workspace programmatically.

You can use these endpoints to invite new team members, retrieve user details, update user access permissions, and manage user roles.

## Key features

- **Role-based access control**: Assign roles such as Admin, Manager, Content Scheduler, and Client.
- **Dynamic permission updates**: Manage granular permissions like billing access, inbox access, and publishing rights.
- **Secure authentication**: Generate and use JSON Web Tokens (JWT) for secure API requests.
